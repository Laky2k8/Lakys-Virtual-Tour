# Laky's Virtual Tour

A virtual tour application to show off a place or places. Suitable for hotels, hostels, restaurants etc.

<br>

## Features
- Interactive map
- Infobox with optional "Open in Google Maps" button
- 360 Panorama viewer
- Regular image viewer
- Full mobile support


# How to host
1. Download and unzip the files.
2. Add your images in the `assets` folder.
3. Modify `locations.json`. You can add as many locations as you want or need.
4. Host the app from a server.
